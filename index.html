This project revolves around building a dynamic frontend interface using modern technologies such as Angular, TypeScript, HTML, and SCSS. Cosmetics Page Shop

The Cosmetics CL website offers an online shopping experience for personal care and cosmetic products.

live preview
Category
Web Development & Design
Start Date
August, 2024

Project Description

Cosmetics Shop Page is an online store simulation focused on providing a user experience that closely mimics a real-life shopping experience.

The platform features a shopping cart where users can add products and see the total price, a search function for easily finding specific items, and a dedicated section for different skin types, guiding users to products tailored to their chosen skin type. Additionally, the website supports English translations to cater to a wider audience.

Design

The product pages are designed to provide a seamless user experience, offering clear visuals and organized information. Additionally, the website features translation options, ensuring accessibility for a broader audience.

Skills

To build this platform, I utilized advanced skills in Angular, TypeScript, HTML, and SCSS, which allowed me to create dynamic components and a responsive design. I also implemented a small API using https://my-json-server.typicode.com/ to simulate a real backend, enabling product data retrieval and management in a structured and efficient manner. This combination of technologies ensures a smooth and efficient browsing experience, tailored to meet the needs of online shoppers.
